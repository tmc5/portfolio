ガスボンベ容器番号認識システムの構築

WPFでWindowsタブレット上で動作するアプリケーションを設計・開発しました。企業との共同開発学内で班長を務めました。

ガス容器の耐圧試験において、人がガス容器番号を試験機に手入力していため、これを自動化するためのシステム開発を行いました。非接触で読み取り可能なRFタグをガス容器に対応させ、RFリーダーで読み取ることによってガス容器番号の認識を行います。

開発過程

開発は以下の流れで行いました。自分はコーディングの一部と企業との連絡係を担当しました。

タブレットでの動作が要件に含まれていたため、Windowsタブレットを用いてアプリケーションの開発を行いました。

  • 要件定義・UI構成
  • アプリ試作
  • 試作品評価
  • UIの改善
  • 完成

Model View ViewModel アーキテクチャ

開発したアプリケーションにはModel View ViewModelアーキテクチャを採用しました。Model View ViewModelアーキテクチャは保守性に優れており、レイアウトを管理するView、データの保持をするViewModel、データ構造・機能を管理するModelの3部に分かれています。データの流れを一方向に制約し、機能ごとにファイルを分割することで優れた保守性を実現しています。
